# Building on Farcaster with Next.js

## Install Telemetry

Next.js has a lovely [Script component](https://nextjs.org/docs/pages/building-your-application/optimizing/scripts).

In your app, you can add this Script component.
You can use the App Router, the Pages Router, or the pre Next.js 13 way.

### App Router

<CodeGroup>

```js
// app/layout.js
import Script from 'next/script'

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>{children}</body>
      <Script src="https://telemetry.castr.guru/script.js" />
    </html>
  )
}
```

```ts
// app/layout.js
import Script from 'next/script'

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>{children}</body>
      <Script src="https://telemetry.castr.guru/script.js" />
    </html>
  )
}
```

</CodeGroup>
