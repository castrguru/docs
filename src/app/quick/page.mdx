# Quick Start

This guide will get you all set up and ready to use the Castr GÜRŲ SDK.
We'll cover how to get started using one of our SDK clients and how to make your first API request.
We'll also look at where to go next to find all the information you need to take full advantage of our powerful REST API. {{ className: 'lead' }}

<Note>

There is NO signup required to start making requests to the Castr GÜRŲ Cloud
— however, we recommend you register a Frame ID from your Cloud dashboard to access full features.
You'll find it under [Settings &raquo; API](#).

</Note>


## Setup Your (Tech) Stack

Before making your first API request, you need to pick which API client you will use.
In addition to good ol' cURL HTTP requests, Castr GÜRŲ offers clients for JavaScript, Python, and Rust.
In the following example, you can see how to install each client.

<CodeGroup>

```js {{ language: 'js', title: 'JavaScript/TypeScript' }}
// Install or run the Castr GÜRŲ JavaScript/TypeScript CLI
npx castrguru@latest
```

```python {{ language: 'python' }}
# Install the Castr GÜRŲ Python library
pip install castrguru
```

```rust {{ title: 'Rust' }}
// Install the Castr GÜRŲ Rust library
cargo install castrguru
```

</CodeGroup>

## Making a REST Request

After picking your preferred client, you are ready to make your first call to the Castr GÜRŲ Cloud.
Below, you can see how to send a GET request to the Profile endpoint to get a list of all your users.
In the cURL example, results are limited to ten users, the default page length for each client.

<CodeGroup tag="GET" label="/v1/profiles">

```bash {{ title: 'cURL' }}
curl -G https://castr.guru/v1/profiles \
  -H "Authorization: Bearer {token}" \
  -d limit=10
```

```js {{ language: 'js', title: 'JavaScript/TypeScript' }}
import { Frame } from 'castrguru'

const frame = await Frame(process.env.APP_FRAME_ID).init()

await frame.ready()
```

```python
from js import Response, Headers
import json

def on_fetch(request):
  data = json.dumps({"hello": "world"})
  headers = Headers.new({"content-type": "application/json"}.items())
  return Response.new(data, headers=headers)
```

```rust {{ title: 'Rust' }}
use serde::{Deserialize, Serialize};
use worker::*;

#[derive(Deserialize, Serialize, Debug)]
struct Json {
  hello: String,
}

#[event(fetch)]
async fn fetch(_req: Request, _env: Env, _ctx: Context) -> Result<Response> {
  let data = Json {
    hello: String::from("world"),
  };
  Response::from_json(&data)
}
```

</CodeGroup>
